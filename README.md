# Stationary Books System

A comprehensive Stationary and Books Management System developed using React.js, Redux Toolkit, Firebase, and Vite. The system supports multiple user roles including Admin, Seller, and Customer.

## Features

### Admin Panel

* Manage Sellers
* Manage Customers
* Monitor Orders
* Dashboard Analytics

### Seller Panel

* Add Products
* Edit Products
* Manage Orders
* Profile Management

### Customer Panel

* Browse Products
* Add to Cart
* Place Orders
* Order History

## Tech Stack

* React.js
* Redux Toolkit
* Firebase
* Vite
* JavaScript
* CSS

## User Roles

### Admin

* Full system control
* Seller management
* Customer management

### Seller

* Product management
* Order processing

### Customer

* Product browsing
* Shopping cart
* Checkout system

## Installation

```bash
git clone https://github.com/Kainaat-Fida/stationary-books-system.git
cd stationary-books-system
npm install
npm run dev
```
